These are the current HTML templates.

You may upload a new one.

Just make sure to use the following pattern:

  • First line of your HTML file should be the <subject> tag. This would contain the subject of the sent mail.
  • Within either the subject or the body, use brackets to define parameters with braces: 
    Hello {fullName}, please make sure you complete your task by {taskDate}
  • You may (and should) use <style> tags to style your HTML, just make sure to escape the styling with double braces 
    
            .logo {{
              text-align: left;
            }}
